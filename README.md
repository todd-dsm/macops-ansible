# macops-ansible

Replacement for the Bash hellscape that is [mac-ops].




## MCP Status

`cursor-agent` is still too new to have any real support. Solve for these:

```json
      "env": {
        "AAP_TOKEN": "<aap-token>",
        "AAP_URL": "https://<aap-url>/api/controller/v2"
```
Then update the `.cursor/mcp.json` file.
